# Security Policy

## Supported versions

TypeScribe is early-stage software. Security fixes target the latest `main` branch and latest tagged release.

## Reporting a vulnerability

Please report security issues privately by opening a GitHub security advisory or contacting the maintainer through the repository owner profile.

Do not include private transcripts, audio files, API keys, or local app storage exports in public issues.

## Local data note

TypeScribe is a local-first desktop app. Transcript drafts, markers, settings, and editor state are saved in local browser/Electron storage on your machine. Deepgram transcription sends the uploaded media file to Deepgram when you choose to transcribe.
