Local-first transcript editor

Timed transcription that still edits like a document.

TypeScribe combines Deepgram transcription, InqScribe-style timed rows, a clean Cara document view, playback sync, markers, and precise range exports in a desktop app.

Build TypeScribe

git clone https://github.com/justelson/typescribe.git
cd typescribe
npm install
copy .env.example .env
npm run desktop

Add a Deepgram API key to .env to run transcription.
